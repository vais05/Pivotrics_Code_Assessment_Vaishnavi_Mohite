## Question 1: Parking Lot System

Designed a simple parking lot system using Object-Oriented Programming.

### Features
- Supports Motorcycle, Car, and Truck.
- Different parking spot sizes.
- Park a vehicle.
- Remove a vehicle.
- Display available parking spots.

## Question 2: Number of Islands

Implemented a Depth First Search (DFS) solution to count the number of islands in a 2D grid.

Time Complexity: O(m × n)

Space Complexity: O(m × n)

---

## Question 3: Linked List Cycle Detection

Implemented Floyd's Tortoise and Hare Algorithm to detect a cycle in a singly linked list.

Time Complexity: O(n)

Space Complexity: O(1)
